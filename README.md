# LLM CLI with Web Search Tool

A simple ChatGPT-like CLI application built with Node.js that supports tool calling and real-time web search.

## Features
- Continuous conversation (session-based memory)
- Tool calling (web search via Tavily)
- Real-time information retrieval
- Modular architecture
- CLI-based interaction (no UI)

## Tech Stack
- Node.js
- OpenAI / Groq API
- Tavily API

## Purpose
This project was built as a practice implementation to understand:
- LLM conversational flow
- Tool calling architecture
- Basic memory handling
